# Wine Recommender

A Flask-based web application for personalized wine recommendations.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/StofflR/WineRecommender.git
   cd WineRecommender
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python app.py
   ```

4. Open your browser and navigate to `http://127.0.0.1:5000`
